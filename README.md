Projeto aplicativo calculadora de IMC realizada em Flutter.

# Flutter IMC-App.

Uma calculadora de índice de massa corporal simples feita com Flutter.
Com este aplicativo, você pode avaliar se está com o peso ideal de acordo com a sua altura.
E dependendo desse resultado, você pode saber se está no peso ideal, abaixo ou acima. 

- For install dependencies:
   $ flutter packages get

   Run the app: 
   $ flutter run
   ```
